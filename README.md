## Project Overview

This project was completed as part of the Future Interns UI/UX Design Internship. The objective was to design and develop a mobile appointment booking application that provides users with a simple and efficient way to schedule appointments with a service provider.

The application includes an onboarding/login screen, service selection screen, appointment booking interface, and appointment confirmation screen. The design focuses on usability, accessibility, and creating a smooth user experience throughout the booking process.

## Objective

To create a user-friendly mobile booking application that allows users to easily select a service, choose an appointment time, and receive confirmation of their booking.

## Screens Included

* Onboarding/Login Screen
* Service Selection Screen
* Appointment Booking Screen
* Appointment Confirmation Screen

## Technologies Used

* HTML5
* CSS3
* Visual Studio Code
* GitHub

## Key Features

* Simple and intuitive user interface
* Mobile-friendly responsive design
* Easy service selection process
* Appointment date and time scheduling
* Clear booking confirmation feedback
* Consistent navigation and layout

## Design Rationale

While developing this application, I focused on creating a booking experience that is simple, efficient, and easy to understand. HTML was used to structure the application screens, while CSS was used to create a visually appealing and consistent user interface.

The onboarding/login screen was designed to provide users with a clear starting point and quick access to the application. The service selection screen allows users to browse available services easily and make selections without confusion. The appointment booking screen was organized to help users choose available dates and times efficiently while minimizing the possibility of booking errors.

The confirmation screen provides immediate feedback after a successful booking, giving users confidence that their appointment has been scheduled correctly. Throughout the project, I focused on usability, responsive design, and consistency to ensure a smooth and enjoyable booking experience across different mobile devices.
